About the Project

SmartContactManager is a full-stack contact management web app that lets you store, edit, and organize your contacts with ease. Built on Spring Boot, Hibernate, and Thymeleaf, it provides:

  • Secure CRUD operations with form validation
  • Search & filter contacts by name or email
  • Profile picture upload and preview
  • Responsive UI powered by Bootstrap 5

Tech Stack

Java 17 Spring Boot Hibernate Thymeleaf MySQL Bootstrap 5
